How to Start Using Vim, the Best Command Line Text Editor

What is Vim?

Vim (Vi IMproved) is a highly configurable text editor built to make creating and changing any kind of text very efficient. Vim is designed for use both from a command-line interface and as a standalone application in a graphical user interface. It is a free and open-source, screen-based text editor program for Unix.

Main Principles

The Vim editor is a modal editor. What it means is that your editing workflow is based on switching between dose modes. The primary goal is to prevent using the mouse as mush as possible (slows your coding).

Primary Modes

  • Normal Mode - By default, Vim start in normal mode. The primary goal of the normal mode is to move around the document, read and enter other modes. Enter it by pressing ESC . Key presses dont work as you will expect. You cant write to the document in the normal mode.
  • Insert Mode - As you expect, in insert mode you can type just like in a regular editor. There are meny ways to enter it, some of theme are:
    • i - swiches to insert mode,
    • o - starts inserting in a new line,
    • A - moves to the end of a line then inserts.
    Exit insert mode by pressing ESC
  • Command Mode - Command mode has lots of command. To enter command mode type : then your desired command. It will apear on the buttom of the screen. The command mode is extremely powerfull and versitile, basically you can do all the things your old editor did and better.
