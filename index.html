What is Vim?

Vim (Vi IMproved) is a highly configurable text editor built to make creating and changing any kind of text very efficient. Vim is designed for use both from a command-line interface and as a standalone application in a graphical user interface.

It is a free and open-source, screen-based text editor program for Unix.
